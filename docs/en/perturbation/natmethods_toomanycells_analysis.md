## 📝 Overview
This repository contains the code to reproduce analyses presented in our Nature Methods manuscript entitled, “TooManyCells Identifies and Visualizes Relationships of Single-cell Clades.” The manuscript can be accessed at [Nature Methods](https://www.nature.com/articles/s41592-020-0748-5). Citation: Schwartz GW, Zhou Y, Petrovic J, Fasolino M, Xu L, Pear WS, Vahedi G, Faryabi RB: TooManyCells Identifies and Visualizes Relationships of Single-cell Clades.  Nature Methods March 2020 Notes: doi: 10.1038/s41592-020-0748-5.
